Xuất CSV để mở bằng Excel. Xuất JSON để sao lưu hoặc nhập lại trên thiết bị khác.

📖 Hướng dẫn lần đầu

① Tạo Service Account (Google Cloud):

• Vào console.cloud.google.com → Chọn project → Bật Google Sheets API

• IAM & Admin → Service Accounts → Tạo mới → Tạo JSON key → Tải file .json về

② Cấp quyền cho Google Sheets:

• Mở Google Sheets → Chia sẻ → Dán email service account (trong file .json) → quyền Editor

③ Nhập vào app (tab GSheet này):

• Dán nội dung file .json vào ô JSON Key

• Nhập Spreadsheet ID (lấy từ URL sheets)

• Nhấn Lưu cấu hình — xong! App tự nhớ mãi mãi.

④ Chuyển sang điện thoại / máy khác:

• Nhấn 📲 Tạo Link cấu hình hoặc 📱 Hiện QR → mở trên thiết bị mới → nhấn Lưu.

• Không cần cấu hình lại từ đầu, không cần GitHub.

🔒 Bảo mật: Config lưu trong bộ nhớ trình duyệt (localStorage), không upload lên bất kỳ server nào. Link chia sẻ chứa private key — chỉ gửi cho chính bạn.
